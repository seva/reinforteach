# Discovery: Unsloth Training Interface

## Interface

Python API only. No CLI. Entry point: `from unsloth import FastLanguageModel`.

---

## DPO

### Dataset format

Required fields: `chosen`, `rejected`. `prompt` is optional but recommended.

**Recommended (explicit prompt, conversational):**
```json
{
  "prompt": [{"role": "user", "content": "..."}],
  "chosen": [{"role": "assistant", "content": "..."}],
  "rejected": [{"role": "assistant", "content": "..."}]
}
```

**Flat string format:**
```json
{"prompt": "The sky is", "chosen": " blue.", "rejected": " green."}
```

### Python API (minimal)

```python
from unsloth import FastLanguageModel, PatchDPOTrainer
from trl import DPOTrainer, DPOConfig

model, tokenizer = FastLanguageModel.from_pretrained(model_name=MODEL_PATH, ...)
model = FastLanguageModel.get_peft_model(model, r=64, lora_alpha=64, ...)

PatchDPOTrainer()
trainer = DPOTrainer(
    model=model, ref_model=None,
    args=DPOConfig(output_dir="outputs", num_train_epochs=3, ...),
    beta=0.1,
    train_dataset=DATASET,
    tokenizer=tokenizer,
)
trainer.train()
```

Key `DPOConfig` params: `loss_type` (default `"sigmoid"`), `beta`, `learning_rate` (default `1e-6`), `output_dir`.

## GRPO

### Dataset format

Required columns: `prompt` (list of message dicts), `answer` (string passed to reward functions).
No `chosen`/`rejected` — preferences are computed at training time via reward functions.

```json
{
  "prompt": [
    {"role": "system", "content": "..."},
    {"role": "user",   "content": "..."}
  ],
  "answer": "ground truth string"
}
```

### Python API (minimal)

```python
from unsloth import FastLanguageModel
from trl import GRPOConfig, GRPOTrainer

model, tokenizer = FastLanguageModel.from_pretrained(...)
model = FastLanguageModel.get_peft_model(model, ...)

trainer = GRPOTrainer(
    model=model,
    processing_class=tokenizer,
    reward_funcs=[reward_fn],
    args=GRPOConfig(
        num_generations=4,     # G: completions generated per prompt
        max_prompt_length=512,
        max_completion_length=512,
        output_dir="outputs",
        ...
    ),
    train_dataset=DATASET,
)
trainer.train()
```

## Output Artifacts

After `trainer.train()`:

```python
# LoRA adapter only (recommended for llama.cpp hot-swap)
model.save_lora("lora_adapter")
# → adapter_model.safetensors + adapter_config.json

# GGUF (for full model swap via POST /models/load)
model.save_pretrained_gguf("model", tokenizer, quantization_method="q4_k_m")
# → model.gguf (or model-Q4_K_M.gguf)

# Merged 16-bit safetensors
model.save_pretrained_merged("model", tokenizer, save_method="merged_16bit")
```

Preferred output path: **LoRA adapter** (`save_lora`) → deploy via `POST /lora-adapters` on llama.cpp server (no model reload, instant swap).
